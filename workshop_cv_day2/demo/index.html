Nazi Symbol Detection

Upload a video file or paste a YouTube URL to scan frames for Nazi symbols. The detection model is a placeholder — replace runDetection() in the code with your own TF.js or YOLO model.

How it works

The scanner extracts frames from your video at 1 frame per second, then runs each frame through an object-detection model. Frames with detections get red bounding boxes drawn over them. Currently the model is a stub that returns no detections — you need to supply your own trained model.

Backend Required

Start the backend server before using this tab:

cd backend && pip install -r requirements.txt && python server.py

The server runs on http://localhost:8000. You need Python 3.8+ with pip.
